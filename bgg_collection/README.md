# BGG Collection

Place your BoardGameGeek CSV export(s) here.

## How to export

1. Go to: https://boardgamegeek.com/geekcollection.php?action=exportcsv&subtype=boardgame&username=YOURUSERNAME
2. Replace `YOURUSERNAME` with your BGG username
3. Save the downloaded CSV file into this folder

Multiple CSV files are supported; all files are merged (duplicates removed by game ID).

Only owned games (own=1) are included in challenge generation.
