# Reading Collection

Place your reading collection CSV exports into the appropriate subfolder:

- **goodreads/** — Goodreads library exports (from https://www.goodreads.com/review/import)
- **storygraph/** — StoryGraph library exports (from your StoryGraph library settings)

Multiple CSV files per source are supported; all files in each subfolder are merged.

The format is auto-detected based on which subfolder the file is in.
